# Math Quiz WinForm

## Overview
Math Quiz WinForm is a simple quiz game built using **.NET 9.0.11**. The application allows users to select the number of questions, difficulty level,
and type of math questions. Users answer questions one by one, and the final score is displayed at the end.

## Features
- Choose number of questions
- Select difficulty level
- Pick type of math questions
- Immediate score display after completing the quiz

## Technologies
- **.NET Version:** 9.0.11
- **Project Type:** Windows Forms Application

## How to Run
1. Open the solution `MathQuizWinForm.slnx` in Visual Studio.
2. Build the solution.
3. Run the `MathQuizWinForm` project.  
4. The game runs immediately after building.
